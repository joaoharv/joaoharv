## About Me

I build end-to-end AI solutions — from data ingestion and feature engineering to model deployment and LLM-powered agents. My focus is on making things work beyond the notebook: structured pipelines, reliable APIs, and systems that connect data to real decisions.

I work with **Python, SQL, FastAPI, Scikit-learn, Pandas** and local LLMs like **Llama 3.1 via Ollama**. I have hands-on experience with ETL pipelines, PostgreSQL, autonomous agents and REST API design. I also translate data into business insight through **Power BI** and structured analytics.

Currently studying **Data Science & AI at FIAP** and holding an **AWS Cloud Practitioner** certification.
